# 📚 Book Vibe

Book Vibe is a React-based book management web application where users can explore books, view detailed information, and organize them into Readlist and Watchlist. This project demonstrates the use of React Router, Data API, and modern UI design using DaisyUI.

## [Live Demo](https://vivebook.netlify.app/)

## 🚀 Features

- Browse all available books
- View detailed information for each book
- Add books to Readlist
- Add books to Watchlist
- Sort and filter books
- Dynamic routing with React Router
- Data loading using React Router Data API
- Responsive UI with DaisyUI

## 🛠️ Technologies Used

- React
- React Router DOM
- React Router Data API
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)

## 🧠 How It Works

React Router handles navigation between pages. Loader functions fetch data before rendering routes. Users can manage books using Readlist and Watchlist. Sorting and filtering improve user experience.

## 🎯 Purpose

This project is built to practice React Router, Data API, state management, and UI design with DaisyUI.
