# 🍵 TeaHouse – Simple Tea Shop Landing Page

🔗 **Repository:** https://github.com/axiomshuvo/PH_workshop  
🔗 **Project Branch:** https://github.com/axiomshuvo/PH_workshop/tree/teahouse  
🔗 **Live Demo:** https://ateahouse.netlify.app/

TeaHouse is a simple and elegant tea shop landing page built using **HTML** and **Tailwind CSS**.  
This project is part of my **PH Workshop practice repository** and is located in the **`teahouse` branch**.

The main goal of this project was to practice **Tailwind CSS utilities** and build a **fully responsive layout** using modern frontend techniques.

---

## ✨ Features

- Clean and minimal UI design
- Fully responsive layout (Mobile, Tablet, Desktop)
- Utility-first styling with Tailwind CSS
- Lightweight and fast loading
- Simple and maintainable HTML structure

---

## 🧠 Learning Focus

While building this project, I practiced:

- Using Tailwind CSS effectively
- Building responsive layouts
- Structuring semantic HTML
- Designing simple landing page sections
- Improving frontend workflow

---

## 🛠 Technologies Used

- **HTML5**
- **Tailwind CSS**

---

## 📱 Responsive Design

The layout adapts to different screen sizes:

- Mobile devices
- Tablets
- Desktop screens

Tailwind’s responsive utilities ensure a smooth user experience across devices.
