# Dream11 Cricket Team Builder

A modern React-based fantasy cricket team selection app where you can build your ultimate Dream11 team using virtual coins. Select players from various categories, manage your budget, and create the perfect squad for cricket matches.

## 🚀 Live Demo

[View Live App](https://mydream11team.netlify.app/)

## 🛠️ Tech Stack

- **Frontend**: React 19
- **Build Tool**: Vite
- **Styling**: TailwindCSS + DaisyUI
- **Icons**: Lucide React (if used)
- **Deployment**: Netlify
