## About This Repository

This repository is a **Programming Hero workshop workspace** where each branch represents either:

- a **practice branch** for learning a topic
- a **starter branch** created from a framework template
- or a **project branch** with a working frontend app and, in several cases, a public live demo

Instead of storing everything in one long main branch history, this repo keeps the learning path separated by branch so each topic or project can be explored on its own.

## Project Highlights

### TeaHouse

A clean tea shop landing page built to practice **responsive layout design** and **Tailwind CSS utility-first styling**.

### Janala

An English vocabulary learning interface that works with Programming Hero APIs for lessons, words, and modal-based word details.

### World On The Go

A countries API project for browsing country information using different query methods such as all countries, code, name, and language.

### Dream11 Cricket Team Builder

A React-based fantasy cricket team builder where users assemble a squad using a virtual budget.

### PAYOO Smart MFS Interface

A digital mobile finance dashboard focused on transactions, balance actions, and responsive UI flow.

### Book Vibe

A book management app built with React Router where users can browse books, view details, and organize lists.
